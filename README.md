# Análise de Performance: RED Canids vs Concorrentes (CBLOL 2025)

   Este trabalho tem como objetivo analisar o desempenho e o comportamento dos canais oficiais da RED CANIDS, VIVO KEYD STARS e PAIN GAMING, afim de compreender suas métricas absolutas, estabilidade, consistência, volume de produção e estratégia de conteúdo ao longo do ano de 2025.
   Após a analise pode-se ser concluido algumas observacões,como por exemplo a falta de * arquitetura estrutural na base da criacão de conteúdos, * forte dependência em recursos externos para a aquisicão de público, * grande repeticão de propostas audio visual e  * insistência em conteúdos prejudiciais ao ecossistema dos canais de YT.
      As ferramentas utilizadas para este projeto foram Python, Pandas, Matplotlib/Seaborn e Excel.
      -/data`: Arquivos CSV com os dados brutos da análise.
-/notebooks`: Scripts em Python utilizados para gerar os insights.
 -`RESUMO_PROJETO.pdf`: Documentação detalhada de 17 páginas com a análise completa.
